2 Factorial

2.1 Basic definition and properties

The factorial of \(n\in\mathbb{N}\) is defined as: \[ n! = \prod\limits_{k=1}^{n} k \]

We can easily deduce that \[ n! = n\cdot (n-1)! \]

2.2 Implementation

Here’s a recursive python implementation

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

Let’s call this function for \(n=4\)

print(factorial(4))
